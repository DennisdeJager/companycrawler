Done for initial scaffold.

## Acceptance Criteria

- FastAPI API exposes website, scan, document, search, model and user endpoints.
- Swagger is available.
- MCP manifest and tool endpoints exist.
- React console supports scan start, progress, tree view, inspector, models, users and docs links.
- Docker compose runs frontend, backend, worker and pgvector.
- Documentation files exist and describe deployment, UX and AI context.
